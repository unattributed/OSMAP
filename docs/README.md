# Documentation

This directory holds public project documentation.

The repository keeps both active baseline documents and explicit placeholder
documents for planned phases of work. Placeholder files should stay
non-sensitive and should clearly state that fuller content is still pending.

Current baseline documents with substantive content:

- `PROJECT_CHARTER.md`
- `ARCHITECTURE.md`
- `SECURITY_MODEL.md`
- `DECISION_LOG.md`

The remaining files in this directory are public placeholders that preserve the
intended documentation map without exposing private planning material.
